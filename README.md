## Proyectos destacados

### ReSet
> Plataforma web de acompañamiento para personas en recuperación de adicciones.

Integra registro emocional diario, sistema de rachas de sobriedad y red de apoyo
ante crisis. Construida con Next.js, NestJS, PostgreSQL y Docker, desplegada en
producción con VPS en DigitalOcean.

`Next.js` `NestJS` `PostgreSQL` `Docker` `TypeScript`

### Hugin & Munin
> Sistema web de gestión de especímenes para el Zoológico Miguel Álvarez del Toro.

Digitaliza y centraliza el registro, consulta y administración de información
operativa y científica de la institución. Proyecto institucional con cliente real.

`Next.js` `Node.js` `PostgreSQL` `Git`
